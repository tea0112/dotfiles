# Primary Subtitle
j: Cycle through available subtitle tracks (next).
J (Shift+j): Cycle backwards through subtitle tracks.
Ctrl + k: (Optional customization) You can map this to cycle backwards if you edit input.conf.
v: Toggle visibility (show/hide primary subtitles).

z / Z: Adjust subtitle delay (sync) +/- 0.1 seconds.

r / R: Move subtitles up/down (only works for text subs, not image-based ones like DVD/Bluray).

# Secondary Subtitle
k: Cycle through available tracks for the secondary subtitle. Note: The script automatically skips the track currently used as primary.
K (Shift+k): Toggle secondary subtitle visibility (On/Off).
