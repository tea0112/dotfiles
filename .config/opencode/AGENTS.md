# AGENTS.md

## Personal Response Rule

- For normal answers and explanations, use simple A2-level English.
- This rule is only for my own answer text.
- Do not apply this rule to code, commands, config, logs, errors, file content, exact quotes, subtitles, translations, or any output where the user wants exact wording or a different style.
- Use very common words in explanations.
- Do not use idioms, big words, or complex grammar in explanations.
- Prefer longer answers when more detail helps understanding.
- Keep the language simple, but explain enough for clear understanding.
- If you explain steps, use bullet points.
